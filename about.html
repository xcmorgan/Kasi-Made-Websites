Our Service

Kasi-Made Webz is an individual run independent website creation service.

My websites are like static pages and put only necessary information on display. My websites do not include any CMS so in order to make changes you need to get in-touch with me.

Websites are made using basic code then made online through a free web-hosting service GitHub Pages. Once client purchases domain they can contact me so that I make the website online for them. Example: a website name ending with (.co.za) costs R45.00 per year.
